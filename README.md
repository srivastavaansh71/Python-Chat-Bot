# SPIDEY - Python Chatbot 

SPIDEY is a simple command-line based chatbot built using Python. It interacts with users, provides basic responses, and creates a personalized greeting based on the current time.

## Features 

* Greets the user according to the current time:

  * Good Morning
  * Good Afternoon
  * Good Evening
  * Good Night
* Takes user's name and creates a personalized interaction
* Simple chatbot conversation system
* Predefined response memory using Python dictionary
* Keyword-based question matching
* Runs continuously until the user types **"bye"**
* Beginner-friendly Python project

## Technologies Used 

* Python
* datetime module
* Functions
* Loops
* Conditional Statements
* Dictionary Data Structure

## How It Works 

1. The program asks the user's name.
2. It checks the current system time and gives a suitable greeting.
3. The user can ask questions to SPIDEY.
4. The chatbot searches for matching keywords and returns a stored response.
5. The conversation ends when the user types "bye".

## Example 

```
Welcome Sir, enter your name : Ansh

Good Evening, Ansh

Hello sir! This is SPIDEY
you can ask me questions, Type 'bye' to exit

Please ask your question: who are you

Bot Response : I am smart Chatbot
```
